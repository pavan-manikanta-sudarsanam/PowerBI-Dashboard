# 📊 Power BI Dashboard – Task Management System

## 📌 Project Overview
This project is an interactive Power BI dashboard designed to analyze and monitor task management performance. It provides insights into task status, priorities, team productivity, and overall project progress using interactive visuals.

## 🛠️ Tools Used
- Power BI
- Power Query
- DAX
- Microsoft Excel

## 📂 Dataset
Task_mangment_system_data.xlsx

## 📈 Dashboard Features
- KPI Cards
- Task Status Analysis
- Priority-wise Analysis
- Team Performance
- Interactive Slicers
- Dynamic Charts
